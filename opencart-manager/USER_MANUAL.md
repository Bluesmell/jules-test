## 3. Getting Started

### Logging In
Access the OpenCart Manager by navigating to `your-opencart-store.com/opencart-manager/`. Use the admin credentials created during the installation process.

### Navigating the Interface
The interface consists of a main navigation bar providing access to all modules. Each module has its own set of tools and reports. Breadcrumbs help you keep track of your location within the application.
